# React Business Card

A simple digital business card for Nikema Prophet. The app presents an avatar, role,
short bio, interests, and social links in a compact layout.

## Features

- Profile header with avatar, name, and role
- About and Interests sections
- Social links with icons
- Vite-powered dev server and build

## Tech Stack

- React
- Vite
- React Icons

## Getting Started

Install dependencies:

```bash
npm install
```

Run the app:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

Lint the project:

```bash
npm run lint
```
